# Repository for article on social class, socioeconomic level of the neighborhood and attachment to society
This repository contains the documentation, processing and analysis code for on working paper titled _Class, spatial inequality and social cohesion: Does concentrated affluence and poverty affect individualsâ€™ attachment to society?_

# Structure of the repository

The repository follows the [IPO protocol](https://lisacoes.com/protocolos/a-ipo-rep/) from the Open Social Science Laboratory (LISA) of the Center for Conflict and Social Cohesion Studies (COES), but add some new folders.

- `docs`: utility documents and code
- `analysis`: analysis scripts to reproduce the main and suplementary analysis of the paper
- `input`: original and processed datasets for the paper (and other inputs)
- `output`: tables, plots and other outputs
- `processing`: processing scripts to reproduce the final datasets
